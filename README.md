# Prompt Sharing Platform

A full-stack web application for creating, sharing, and discovering creative prompts.

## Features

-   **User Authentication:** Secure user signup and login functionality.
-   **Prompt Management:** Create, view, and manage your own prompts.
-   **Discover Prompts:** Browse and search for prompts created by other users.
-   **Feedback System:** Like and dislike prompts to provide feedback.
-   **View Counts:** Track the popularity of prompts with view counts.

## Tech Stack

-   **Frontend:** [Next.js](https://nextjs.org/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Tailwind CSS](https://tailwindcss.com/)
-   **Backend:** [FastAPI](https://fastapi.tiangolo.com/), [Python](https://www.python.org/), [SQLAlchemy](https://www.sqlalchemy.org/)
-   **Database:** [PostgreSQL](https://www.postgresql.org/)
-   **Containerization:** [Docker](https://www.docker.com/)
-   **Process Management:** [Supervisor](http://supervisord.org/)

## Getting Started

### Prerequisites

-   [Docker](https://www.docker.com/get-started)
-   [Docker Compose](https://docs.docker.com/compose/install/)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/your-repository.git
    cd your-repository
    ```

2.  **Build and run the application with Docker:**

    ```bash
    docker build -t prompt-app .
    docker run -p 3000:3000 -p 10000:10000 prompt-app
    ```

    *Note: The backend runs on port 10000, and the frontend on port 3000.*

3.  **Access the application:**
    -   Frontend: [http://localhost:3000](http://localhost:3000)
    -   Backend API Docs: [http://localhost:10000/docs](http://localhost:10000/docs)
